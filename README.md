# NNMintFurniture
### 重学 Swift ，练习项目中一些常用的功能。
## 持续更新中...

## 项目中使用到的第三方库：
 - Alamofire (4.3.0) - 网络请求 
 - Kingfisher (3.3.4) - 缓存图片 
 - SVProgressHUD (2.1.2) - 提示框
 - SnapKit (3.1.2) - 设置约束 
 - SwiftyJSON (3.1.4) - 解析 JSON 数据
